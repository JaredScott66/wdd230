Hick's Law

Interaction Design

interaction-design.org

Hicks law is a metric by which you follow in order to limit or control the number of choices to be made. If there are too many the user could end up with choice paralisis. Or just leaving when not quickly finding what they are looking for. The screen shot shows this principle with how few elements there are. you can quickly find the next step in what you are looking for.

Fitt's Law

Interaction Design

interaction-design.org

Fitts law, is rule set that helps to determine optimal spacing and sizing between different page elements. The idea that smaller elements take more focus, less space and more time to click. This leads to more time used and is more likely to become frustrating. If an element is too large it can have similar issues.
